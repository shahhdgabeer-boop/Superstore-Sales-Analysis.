# 📊 Superstore Business Analysis

This project provides a comprehensive analysis of sales trends, regional profitability, and the impact of discounts using **Python**, **Excel**, and **Power BI**.

## 🔍 Key Insights from My Report
* **The "Discount Trap":** I discovered that profit declines sharply when discounts exceed **20%**. Discounts of **50%+** lead to financial losses.
* **Profitability Leader:** The **West Region** is the most profitable area for the store.
* **Top Category:** **Technology** generates the highest sales volume.

## 🛠 Tools Used
* **Python (Pandas):** Used for data cleaning and analysis (see `Superstore_Analysis (1).ipynb`).
* **Excel:** Initial data exploration and auditing.
* **Power BI:** Built interactive dashboards to visualize KPIs.
